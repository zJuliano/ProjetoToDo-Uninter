## Descrição do Projeto

Este projeto implementa uma lista de tarefas (To-Do List) com funcionalidades básicas de cadastro e gerenciamento de tarefas no navegador.

## Funcionalidades

- Adicionar tarefa com título, descrição e prioridade
- Editar tarefa existente
- Marcar tarefa como concluída/desmarcar
- Excluir tarefa
- Persistir tarefas no localStorage

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (DOM e localStorage)
